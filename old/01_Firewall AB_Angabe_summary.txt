**Firewalls**

* Definition: A firewall is a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
* Types:
	+ Desktop Firewall (Software): protects individual devices
	+ Network Firewall (Hardware): protects entire networks
* Purpose: to prevent unauthorized access, denial of service attacks, and other malicious activities

**Paketfilter (Packet Filtering)**

* Definition: a type of firewall that filters packets based on specific criteria such as source and destination IP addresses, ports, and protocols.
* Types:
	+ Stateless Packet Filter: works independently of previous packets; requires multiple rules for TCP connections
	+ Stateful Inspection Firewall: keeps track of connection status and allows follow-up packets to be associated with the existing connection

**Comparison of Stateless and Stateful Packet Filtering**

|  | Stateless Packet Filter | Stateful Inspection Firewall |
| --- | --- | --- |
| Advantages | High security, low functionality | High functionality, moderate security |
| Disadvantages | Requires many rules, slow performance | Requires less rules, but may be vulnerable to attacks |

**Firewall Policy**

* A firewall policy is a set of rules that determines how packets are handled by the firewall.
* There are two main types:
	+ Black-List: denies all traffic unless explicitly allowed
	+ White-List: allows only explicitly allowed traffic

**Network Firewall**

* Definition: a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
* Purpose: to prevent unauthorized access, denial of service attacks, and other malicious activities.
* Characteristics:
	+ Centralized administration
	+ Reduced administrative effort
	+ More secure than desktop firewalls

**Conclusion**

* Firewalls are an essential component of network security.
* There are different types of firewalls, including desktop and network firewalls.
* Packet filtering is a key feature of firewalls, with stateless and stateful packet filters offering different advantages and disadvantages.
